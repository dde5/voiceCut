# 全功能純前端音檔編輯工作站 (VoiceCut)

這是一個基於 Web Audio API 開發的單一檔案音訊編輯工具，無需安裝任何軟體或後端伺服器，直接在瀏覽器中即可運行。它提供了強大的音訊分析、剪輯、混音與匯出功能。

## ✨ 主要功能

### 1. 核心編輯
- **波形可視化**：直觀的主音軌波形顯示，支援縮放與平移。
- **精確選取**：可透過拖曳或輸入時間數值來精確選取音訊範圍。
- **非破壞性/破壞性編輯**：
  - **剪除**：移除選取的音訊段落。
  - **保留**：僅保留選取的範圍，移除其他部分 (Crop)。
  - **增強**：調整選取範圍的音量 (增益/衰減)。
  - **提取**：將選取範圍提取為獨立的片段 (Clip)。

### 2. 片段管理與混音
- **片段列表**：管理所有提取或匯入的音訊片段。
- **片段編輯**：提供獨立的 Modal 視窗，可對單一畫面進行精細編輯（剪除、增強等）。
- **混音軌 (Secondary Track)**：
  - 支援將片段拖曳或指定時間點安插至混音軌。
  - 可同時播放主音軌與混音軌進行預覽。
- **片段合併**：勾選多個片段將它們合併為一個新片段。

### 3. 智慧尋找 (Smart Search)
- **依音量**：自動偵測並標記超過特定音量的區段 (可用於去除靜音)。
- **依頻率**：基於零交越率 (Zero-crossing rate) 概略偵測高頻或低頻區段。
- **批次導航**：快速在偵測到的區段間切換。

### 4. 匯入與匯出
- **支援格式**：支援瀏覽器能解碼的所有音訊格式 (MP3, WAV, FLAC, M4A, MP4 等)。
- **匯出格式**：
  - **MP3** (使用 lamejs 編碼)
  - **WAV**
- **下載功能**：可單獨下載片段，或下載完整的混音結果。

### 5. 即時互動與視覺增強 (New!)
- **游標播放**：在主音軌任意處點擊，即可從該點開始播放 (點選處會顯示播放按鈕)。
- **播放暫停/恢復**：暫停後，播放位置會停留在暫停點，再次播放則從該點繼續 (Resume from position)。
- **播放速度控制**：提供 0.1x - 10.0x 的無段變速播放，並內建常用倍速按鈕。
- **視覺同步**：
  - **同步指示器**：滑鼠游標在混音軌移動時，主音軌會顯示對應位置的綠色指示線。
  - **常駐播放頭**：紅色播放頭 (Playhead) 在暫停時會保留在畫面上，不會消失。
  - **時間標籤**：播放頭與游標指示線上方皆有即時的時間碼標籤。
- **混音軌管理**：混音軌上的片段現在支援右鍵選單移除。

## 🚀 如何使用

1. **開啟檔案**：直接用瀏覽器 (建議 Chrome, Edge, Firefox) 開啟 `VoiceCut.html`。
2. **載入音訊**：點擊「選擇檔案」載入您要編輯的音訊檔。
3. **開始編輯**：
   - 在主波形圖上拖曳選擇範圍。
   - 使用下方控制列進行「播放」、「剪除」、「提取」等操作。
   - 提取的片段會出現在右側列表 (或點擊收合按鈕展開)。
4. **混音與匯出**：
   - 在片段列表中選取片段 (Radio button)，然後點擊「混音軌」的任意位置將其安插。
   - 完成後點擊「合併並匯出」即可下載成品。

## 🛠️ 技術細節

- **架構**：純 HTML5 + CSS3 + JavaScript (ES6+)。
- **音訊處理**：Web Audio API (AudioBuffer, ScriptProcessor, OfflineAudioContext)。
- **外部依賴**：`lamejs` (透過 CDN 引入) 用於 MP3 編碼，無需後端轉檔。
- **資料隱私**：所有運算皆在本地端執行，音訊檔案不會上傳至任何伺服器。

## ⚠️ 注意事項

- 由於是純前端處理，對於極大的音訊檔案 (>300MB) 可能會消耗較多記憶體。
- 建議使用桌面版瀏覽器以獲得最佳體驗。
